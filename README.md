# Система массового обслуживания для обработки заявок
## *Выполнил: Чугунков В. С.*

## :information_source: Цель: 
- Создать **систему массового обслуживания (СМО)** по обработке заявок, используя язык **Python**.
- У каждой заявки есть **время, которое необходимо потратить на ее обработку**, находящееся в промежутке **[a, b]**. Исходя из этого времени нужно создать **систему**, которая бы наиболее быстро обрабатывала заявки, поступающие в реальном времени c промежутком друг между другом в **[c, d]**

## :gear: Структура системы:
Система будет состоять из 3 основных функций:
1. **MassServiceSystem(receiption_queue, processing_queue, a, b, c, d)**.
    - *Основная функция, запускающая работу системы. В ней происходит распределение новых заявок и управление очередью заявок*
2. **ApplicationProcessing(max_time, current_time)**
    - *Обрабатывает заявки*
3. **ApplicationReceiption(current_time, last_app_time, c, d)**
    - *Функция приема новой заявки. Если обработчик свободен отправляет заявку на обработку, если обработчик занят - отправляет в очередь*

По завершении работы СМО основная функция возвращает счетчики обработанных и необработанных заявок, а так же список заявок, которые не успели обработать
> **Подробное описание можно найти в отчете**

## :white_check_mark: Итоги
Созданная система позволяет стабильно и исправно обрабатывать заявки, поступающие в реальном времени
